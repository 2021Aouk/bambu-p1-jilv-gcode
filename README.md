# bambu-p1-jilv-gcode


<h1 style   风格   风格="font-size: 40px;">急律gcode</h1>



## 项目介绍：

拓竹p1的急律gcode，冲刷浪费降至下限、竹屎体积减小、不用调z补偿，支持多色、tpu，不磨嘴、无清理线、精简步骤、快速开打、0维护.已经过半年连续测试

万一万一需要自己调试zoffset，需要有打印机调平经验。不良反应：不可避免的概率性没擦干净嘴(概率和官方差不多)

建议观看视频BV1ddS2YREu6

## 使用注意


;原理是打完再清理喷嘴，节省掉下次开打清理的时间，所以首次运行前擦干净喷嘴！！

;首次用擦不干净喷嘴为正常现象，循环两三次就好

;开始和结束g代码都在同一个文件里，分别复制进切片的开始和结束gcode
