# bambu-p1-jilv-gcode


<h1 style   风格   风格="font-size: 40px;">急律gcode</h1>



## 项目介绍：

拓竹p1的急律gcode，精简步骤、快速开打、不磨嘴、无清理线（默认有）、0维护、冲刷浪费降至下限、竹屎体积减小（更少倒屎盒）、不用调z补偿，支持多色、tpu.已经过1年连续测试

建议观看视频BV1ddS2YREu6

## 使用注意


;急律gcode原理是打完再清理喷嘴，节省掉开打前清理的准备时间，所以首次运行前擦干净喷嘴！！

;首次用擦不干净喷嘴为正常现象，热端开300度用镊子清理干净喷嘴再使用急律gcode，最多循环两三次就好了

;如果打印中途取消会导致擦嘴代码没执行，请手动把热端开300度用镊子清理干净喷嘴

;如果你不想深究原理，请使用默认带擦嘴线的版本，它合适每次打印换材料的玩家，冲刷擦嘴保证材料干净但是废料。

;如果你是单材料连续打印的玩家，可以平时使用不冲刷无擦嘴线的版本，它省下了冲刷的料，您在切换耗材时用一次默认版本急律gcode即可。

;开始和结束g代码都在同一个文件里，分别复制进切片的开始和结束gcode

;交流群：522185942，欢迎反馈问题、合作开发~

;风险须知：不可避免的概率性没擦干净嘴(概率比官方低)。如果你的机器公差过大需要自己补偿z归零点，可使用orca补偿-0.05，或g代码里改。
